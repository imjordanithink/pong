--PONG--

-STARTUP INSTRUCTIONS-
  Click on the file labled "pong" or "pong.py". It should have a floppy disk graphic attached to it. Make sure to keep this file within the folder that it was installed in.
  The folder can be moved and renamed.


-GAME INSTRUCTIONS-
  - L:
    Tab = up
    CapsLock = down

  - R:
    " \ " = up
    Return/Enter = down

  - GENERAL (on title screen):
    " space " = start

-GAME RULES-
  - Point Gain:
    "When player1 is unable to bounce the ball, player2 will recieve a point. The reverse is also true."
  - Win Param:
    "1st to 10 points wins"
